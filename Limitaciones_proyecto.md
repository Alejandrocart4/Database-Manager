# Database Manager Desktop – Qt \& MariaDB



## Limitaciones del proyecto



* MariaDB 12 soporta paquetes únicamente en modo Oracle como mecanismo de compatibilidad. Dado que no forman parte del modelo estándar de MariaDB/MySQL, se documentan como no aplicables.



* MariaDB maneja tablespaces internamente a nivel del motor InnoDB y del sistema de archivos. No existe un objeto lógico administrable mediante SQL para crear, listar o modificar tablespaces, por lo que se documenta como no aplicable.
